(3) 生成ssh密钥文件

此步生成两个文件。id_rsa为私钥文件。id_rsa.pub为公钥文件。

(4) 查看并复制公钥文件

复制公钥文件中的所有内容。

(5) 将公钥内容上传至Github设置中，实现Github与Hexo同步

打开GitHub_Settings_keys界面，新建new SSH key.
标题随意，如hexo keygen。
key文本框内，将复制的公钥内容粘贴进去。
以上过程为设置公钥，以保障github识别推送的用户为合法用户，实现Github与Hexo同步。
